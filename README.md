# Пользователи — Infotecs

Приложение для отображения и управления данными пользователей с использованием [DummyJSON API](https://dummyjson.com/docs/users).

## Функциональность

- **Таблица пользователей** — фамилия, имя, отчество, возраст, пол, телефон, email, страна, город
- **Сортировка** — по ФИО, возрасту, полу и телефону (3 состояния: по возрастанию ↑, по убыванию ↓, без сортировки ⇅)
- **Фильтрация** — по ФИО, возрасту, полу, номеру телефона
- **Постраничная навигация** — 10 пользователей на страницу
- **Модальное окно** — подробная информация о пользователе по клику на строку (ФИО, возраст, адрес, рост, вес, телефон, email, аватар)
- **Изменение ширины колонок** — перетаскивание границ колонок (минимум 50px)
- **Обработка ошибок** — при сбоях HTTP-запросов с возможностью повторить

## Технологии

- React 19
- TypeScript
- Vite
- Fetch API

## Разработка

```bash
npm install
npm run dev
```

Откройте http://localhost:5173

## Сборка

```bash
npm run build
npm run preview
```

## GitHub Pages

Для деплоя на GitHub Pages:

1. В настройках репозитория: **Settings → Pages → Build and deployment**
2. Выберите **GitHub Actions** как источник

Приложение будет доступно по адресу: `https://<username>.github.io/infotecs/`
